# Full-stack online bookshop App


### `App Feature:`

1. User can order their favorite books and check their previous order. \
2.Admin can add books and manage orders. \
3.Login with Google. \


### `Front-End technology use:`

1.React \
2.Bootstrap \
3.React-Router \
4.Firebase \
5.React-hook-form \
6.Axios

### `Back-End technology use:`
1.Node Js \
2.Express \
3.Body-parser \
4.Cors \
5.MongoDB \
6.Nodemon \
7.Dotenv
